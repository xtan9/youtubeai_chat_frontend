# Gradient tokens

Brand gradients are first-class tokens. Unlike colors / shadows / radius,
**Tailwind 4 has no native `--gradient-*` namespace** — declaring
`--gradient-brand-primary` in `@theme` does NOT auto-generate a
`.bg-gradient-brand-primary` utility. We bridge the gap with the
[`@utility` directive](https://tailwindcss.com/docs/adding-custom-styles#adding-custom-utilities):
`@theme` defines the gradient as a CSS variable, and an `@utility` rule
wraps it as `.bg-gradient-{name}` so consumer code stays clean.

```css
/* In app/globals.css */
@theme {
  --gradient-brand-primary: linear-gradient(
    to right,
    var(--color-purple-500),
    var(--color-cyan-500)
  );
}
@utility bg-gradient-brand-primary {
  background-image: var(--gradient-brand-primary);
}
```

## Brand gradients

| Token | Utility | Stops | When to reach for |
|-------|---------|-------|-------------------|
| `--gradient-brand-primary` | `bg-gradient-brand-primary` | purple-500 → cyan-500 | Primary CTA backgrounds, brand accents (rest state) |
| `--gradient-brand-primary-hover` | `bg-gradient-brand-primary-hover` | purple-600 → cyan-600 | Same CTA in hover/active state |
| `--gradient-brand-accent` | `bg-gradient-brand-accent` | purple-500 → pink-500 → cyan-500 | Animated halos, hero borders, emphasis surfaces |
| `--gradient-brand-soft` | `bg-gradient-brand-soft` | purple-500 (20% alpha) → cyan-500 (20% alpha) | Subtle backdrop glow behind brand-primary surfaces |
| `--gradient-error` | `bg-gradient-error` | red-500 → orange-500 | Error CTAs, destructive emphasis |
| `--gradient-success` | `bg-gradient-success` | emerald-500 → teal-500 | Success state callouts, completed indicators |

Each utility is a normal Tailwind class — combine with hover/dark
variants like any other utility:

```tsx
<button className="bg-gradient-brand-primary hover:bg-gradient-brand-primary-hover">
  Summarize
</button>

{/* Dark-mode-only halo */}
<div className="hidden dark:block bg-gradient-brand-accent blur-sm rounded-3xl" />

{/* Soft backdrop behind a primary surface */}
<div className="absolute inset-0 bg-gradient-brand-soft blur-xl rounded-2xl" />
```

## Composition

Brand-primary is the canonical CTA gradient. Brand-accent (with the pink
mid-stop) is reserved for animated halos and emphasis surfaces — use
sparingly. Brand-soft sits behind brand-primary surfaces in dark mode to
warm the backdrop.

## Direction

The current set goes left-to-right (`to right`). If a future surface
needs a top-to-bottom or radial variant, add a sibling token (e.g.,
`--gradient-brand-primary-vertical`) rather than introducing inline
arbitrary values.

## Why gradients aren't `--color-*`

A gradient is a `background-image`, not a `background-color`. Tailwind's
`--color-*` namespace generates `background-color`, `border-color`, and
`color` (text) utilities — none of which apply to gradients. The
`@utility` indirection sidesteps the namespace mismatch.
