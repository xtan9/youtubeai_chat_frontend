# Typography tokens

Typography lives at two levels: the **base scale** (Tailwind 4's default
`--text-{xs..9xl}`) and the **semantic scale** (display, h1–h6, body
sizes, caption) defined in `app/globals.css` `@theme`.

Reach for the semantic scale when you can — the names carry intent and
swap in a future brand-redesign without touching consumer code. Drop down
to the base scale only when no semantic alias fits.

## Semantic scale

| Token | Utility | Size | Line height | Tracking | Weight | When to reach for |
|-------|---------|------|-------------|----------|--------|-------------------|
| `--text-display` | `text-display` | 3.75rem (60px) | 1 | -0.025em | 700 | Hero headlines (one per page) |
| `--text-h1` | `text-h1` | 3rem (48px) | 1.1 | -0.025em | 700 | Page title |
| `--text-h2` | `text-h2` | 2.25rem (36px) | 1.2 | -0.02em | 700 | Section heading |
| `--text-h3` | `text-h3` | 1.875rem (30px) | 1.25 | -0.015em | 600 | Subsection heading |
| `--text-h4` | `text-h4` | 1.5rem (24px) | 1.3 | — | 600 | Card heading |
| `--text-h5` | `text-h5` | 1.25rem (20px) | 1.4 | — | 600 | Group label |
| `--text-h6` | `text-h6` | 1.125rem (18px) | 1.5 | — | 600 | Inline emphasis heading |
| `--text-body-lg` | `text-body-lg` | 1.125rem (18px) | 1.6 | — | inherit | Lede paragraph, prominent body |
| `--text-body-md` | `text-body-md` | 1rem (16px) | 1.6 | — | inherit | Default body |
| `--text-body-sm` | `text-body-sm` | 0.875rem (14px) | 1.5 | — | inherit | Secondary body, dense UI |
| `--text-body-xs` | `text-body-xs` | 0.75rem (12px) | 1.5 | — | inherit | Micro UI text |
| `--text-caption` | `text-caption` | 0.75rem (12px) | 1.4 | 0.025em | inherit | Image caption, supportive label |

Each token sets size + line-height + (optionally) letter-spacing +
font-weight in one utility — paired Tailwind 4 sub-properties keep them
coherent.

```tsx
<h1 className="text-display">Understand YouTube videos</h1>
<h2 className="text-h2">In a fraction of the time</h2>
<p className="text-body-md">Body copy uses the body-md alias.</p>
<span className="text-caption">Image alt-text or footnote.</span>
```

## Base scale (Tailwind 4 defaults)

`text-xs` through `text-9xl` and the `font-{thin..black}` weight ladder
are also available. They fit when no semantic alias applies (e.g., a
visual-only marketing flourish that doesn't fit any heading level).

## Tracking & leading

Tailwind 4 ships `tracking-{tighter,tight,normal,wide,wider,widest}` and
`leading-{tight,snug,normal,relaxed,loose}`. Use the semantic typography
tokens above first — most need-cases are already covered.

## Font families

`--font-sans` (Geist Sans) and `--font-mono` (Geist Mono) come from
`app/layout.tsx` via `next/font/google`. Brand redesign is out of scope
for B.
